# W Eiji 論 — 二人のワタナベエイジ

神経科学者・渡辺英治と現代美術家・渡辺英司——同姓同名の二人の思考構造をネットワークとして可視化し、その交差と分岐を分析するプロジェクト。17冊の手書きノートと学術論文から概念を抽出した個別分析、二つの回路を統合したグラフ分析、そして両者に共通する「分類体系から対象を切り抜いて解放する」操作の構造と正反対の態度の共存を論じる「切り抜きの美学」の三部構成。
