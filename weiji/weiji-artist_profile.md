## II. 芸術活動の核心テーゼ

### 根本的問い

> **「命名が事物を存在化させるのか」**

私たちは世界を「見る」前に、すでに「名付けている」。名前を知らないものは、認識の網の目からすり抜けていく。渡辺の作品は、この認識の網の目そのものを可視化し、その恣意性と脆弱性を露呈させる。

### 一貫した手法

> **「身近な素材を使って、既存の概念や認識を問い直す」**

| 操作 | 説明 |
|------|------|
| 反転・転覆 | 常識的な順序や関係性を逆転 |
| 転用 | 日常物の文脈を変える |
| ずらし | 期待される位置からの微妙な逸脱 |
| 宙づり | 意味の確定を保留する |
| 言葉遊び | 言語と視覚の関係への揺さぶり |

## III. 代表作シリーズと哲学的背景

### 1. 《蝶瞰図》(Butterfly's-Eye View)

#### 基本情報
| 項目 | 内容 |
|------|------|
| 開始年 | 2009年〜継続 |
| 素材 | 蝶図鑑からの切り抜き |
| 技法 | 空間インスタレーション |
| 主要展示 | 越後妻有トリエンナーレ2009（福武ハウス）、ふじ・紙のアートミュージアム（2018-2019） |

#### 制作プロセスの「逆転」

通常の図鑑制作：
```
自然界の蝶 → 採集 → 標本化 → 図鑑への収録 → 分類・命名
```

渡辺の作品プロセス：
```
図鑑 → 切り抜き → 名前からの切断 → 空間への解放 → 新たな「自然」の創出
```

> **「普通は外の蝶を採集して図鑑に収録」→「図鑑から蝶を解き放つ」**

#### 荘子「胡蝶の夢」との哲学的対話

**荘子の問い：**
> 昔者、荘周夢に胡蝶と為る。栩栩然として胡蝶なり。
> （昔、荘周は夢の中で蝶になった。ひらひらと飛び、蝶そのものであった）

夢と現実、自己と他者の境界はどこにあるのか？

**渡辺の問い：**
> 「蝶の名前（人間が分別すること）が先か、蝶（事物そのもの）が先か」
> 「命名が事物を存在化させているのか」

両者に共通するのは「物化」（ぶっか）——万物は変化し続け、区別は相対的なものに過ぎないという思想。

#### 「蝶瞰図」という造語の意味

| 鳥瞰図 | 蝶瞰図 |
|--------|--------|
| 上空から見下ろす視点 | 蝶の目線から見上げる視点 |
| 全体を俯瞰する支配的視点 | 断片の集積から全体を見る視点 |
| 人間中心の認識 | 非人間的知覚への想像 |

天井一面に配置された蝶を見上げる鑑賞者は、自らが「見られている」という逆転した視点を経験する。

#### 越後妻有アートトリエンナーレ公式解説
> "The butterfly figures covering the room's ceiling, were cut from butterfly reference books. By cutting the figures out of the pages in the reference books, each butterfly is severed from the names given to them. Each butterfly can be seen as an icon of an opened book, so from one reference book numerous small books spread out like a spectrum. With the vertically elevated point of view, amazement surfaces towards the act of naming and classifying things."

### 2. 《名称の庭》(Garden of Names)

#### 基本情報
| 項目 | 内容 |
|------|------|
| 開始年 | 1992年〜継続（30年以上） |
| 素材 | 植物図鑑・キノコ図鑑からの切り抜き、針金 |
| 技法 | 床面・壁面・天井への空間配置 |
| 主要展示 | あいちトリエンナーレ2010、箱根彫刻の森美術館2014、ARTZUID 2017、東京ビエンナーレ2025 |

#### 「名前を知る」ことと「存在を認識する」こと

> 「散歩道で出合った植物やキノコの名前を知り得たとき、初めて目の前のものが自分自身の中で名指しできるものとして現れ、個性として認識されるとも言えます」

#### ソシュール言語学との呼応

- **シニフィアン（記号表現）：** 「蝶」という音声・文字
- **シニフィエ（記号内容）：** 私たちが想起する蝶のイメージ

ソシュールは、言語が世界を分節し、認識可能にすると主張した。渡辺の作品はこの理論を視覚的に体験させる装置である。

例：日本語では「蝶」と「蛾」を区別するが、フランス語では両方「パピヨン」と呼ぶ。この違いは自然界の区別ではなく、言語による分節の結果である。

#### 「人間的自然」の三層構造

> 「そうして確認された図版群を切り抜き、再度、異なる空間に『名付けられた庭』として再現するとき、私たちは新たな人間的自然を目の当たりにする」

1. **第一の自然：** 命名される以前の、純粋な「もの」としての植物
2. **第二の自然（図鑑）：** 記録され、分類された植物
3. **第三の自然（渡辺の庭）：** 図鑑から切り取られ、新たな空間に再配置された植物

#### 分類学への問い

リンネの二名法に始まる近代分類学は、自然界を体系的に整理する「啓蒙の知」の象徴。渡辺は図鑑という「知の体系」を素材として用いることで、その体系自体を脱構築する。

| 分類学の前提 | 渡辺の作品が示すもの |
|--------------|----------------------|
| 種は明確に区別できる | 区別は人間の認識による構築物 |
| 命名は客観的な同定に基づく | 命名は存在を創出する行為 |
| 図鑑は自然を正確に再現する | 図鑑は自然の「模倣」であり、作品はその「模倣の模倣」 |

### 3. 《Merman/おとこの人魚》

#### 基本情報
| 項目 | 内容 |
|------|------|
| 制作年 | 2020年 |
| 素材 | 絵画（テキストベース） |
| 展示 | ケンジタキギャラリー個展 |

#### コンセプト：言葉だけで像を生成する

絵の具や写真を一切使わず、**テキストのみで視覚的イメージを喚起する**絵画シリーズ。

**他シリーズとの対照：**

| 蝶瞰図・名称の庭 | Merman |
|------------------|--------|
| 図像から名前を切り離す | 名前から図像を生成させる |
| 視覚→言語 の切断 | 言語→視覚 の生成 |

#### 「人魚」というモチーフの選択

人魚（mermaid/merman）は、人間と魚という異種の混合体であり、境界の曖昧さを象徴。

- 人間でも魚でもある／人間でも魚でもない
- 海でも陸でも生きられる／どちらにも完全には属さない
- 美しくも危険な存在

荘子の「胡蝶の夢」における荘周と蝶の関係性——自己と他者、現実と夢の境界の揺らぎ——と通底する。

## IV. 哲学的フレームワーク

### 1. プラトン「イデア論」との関係

プラトンは、現実世界の事物は「イデア」（完全な原型）の不完全な模倣に過ぎないと説いた。

**プラトン的階層と渡辺の作品：**
```
イデアとしての蝶（完全・永遠）
      ↓
実在の蝶（不完全な模倣）
      ↓
図鑑の蝶（模倣の模倣）
      ↓
渡辺の作品の蝶（模倣の模倣の模倣？ or 解放された新たな存在？）
```

### 2. ハイデガー「存在と命名」

ハイデガーは『芸術作品の根源』で、芸術作品が「真理」を開示すると論じた。渡辺の作品は、命名という行為が存在を「開示」すると同時に「覆い隠す」という二重性を示す。

### 3. デリダ「脱構築」

ジャック・デリダの脱構築は、二項対立の内部崩壊を示す戦略。渡辺の作品が脱構築する対立：

- 自然 vs 人工
- オリジナル vs 複製
- 存在 vs 命名
- 全体 vs 断片

### 4. 荘子「万物斉同」

> 「万物は斉しく同じ」——この世の全ての物事は人間の視点からは別に見えても、天地自然の道から見ればまったく同じ

渡辺の作品はこの東洋哲学を視覚的に体現する。

## V. 制作手法と時間性

### 素材
- 図鑑（蝶図鑑、植物図鑑、キノコ図鑑）— 安価な中古品も使用
- 針金、テープ
- 大理石、木
- 油彩、墨
- ダイス、人工芝、障害物競走用ハードル

### 「安価な素材」と「深遠な問い」のコントラスト

渡辺は高価な素材や特殊な技術を使わない：
- 中古の図鑑（数百円〜数千円）
- ハサミとワイヤー
- 市販の画材

この「安価さ」は意図的な選択であり、「芸術とは何か」という問いへの一つの回答。

### 時間の蓄積

- 何百もの蝶を一つ一つ切り抜く
- 何千もの植物図版を切り取りワイヤーで吊るす
- 30年以上にわたって同じシリーズを展開し続ける

《名称の庭》は1992年から継続中であり、「完成」という概念が適用されない。これは以下を示唆する：
- 知識の体系は常に未完成である
- 分類は永遠に終わらない作業である
- 存在の認識は継続的な行為である

## VIII. 神経科学者・渡辺英治との比較分析

### 二人の「英治」の思考的交差点

| 観点 | 現代美術家・渡辺英司 | 神経科学者・渡辺英治 |
|------|------------------|----------------------|
| 蝶 | 存在論的メタファー（胡蝶の夢） | 運動錯視の研究対象（Rotating Snake） |
| 知覚 | 命名による認識の変容 | 予測符号化による知覚生成 |
| 境界 | 夢と現実、自己と他者 | 意識と無意識、予測と誤差 |
| 沈黙 | 言葉なき存在の探求 | 「意識は基本的にsilent」 |
| 時間 | 蓄積と配置の時間、30年の継続 | 100msの知覚窓 |
| 方法 | 切り抜き・配置・反転 | 実験・数理モデル・PredNet |

### 「予測と誤差」の芸術的等価物

**神経科学者・渡辺英治の予測符号化理論：**
> 脳は世界を予測し、予測誤差を最小化するシステムである

**現代美術家・渡辺英司の作品：**
> 鑑賞者の「予測」（図鑑＝知識の体系）を裏切り、「誤差」（切り抜き＝脱文脈化）を通じて新たな認識を生成する

両者は異なるアプローチで同じ問いに向かっている：**人間はいかにして世界を認識するのか？**

### 共鳴可能なテーマ

1. **知覚と認識**: 何かを「見る」とはどういうことか
2. **分類と命名**: 名前がつくと存在が変わるか
3. **蝶のモチーフ**: 運動知覚（錯視）と存在論（胡蝶の夢）
4. **反転の思考**: 既存の枠組みを逆転させる発想
5. **集積と創発**: 断片の蓄積が生む全体性
6. **沈黙と意識**: 言語化されない認知過程

## X. 鑑賞者が経験するプロセス

渡辺の作品を前にした鑑賞者は、以下の経験をする：

1. **驚き（Surprise）**: 図鑑から切り取られた蝶の大群に圧倒される
2. **認識（Recognition）**: これが「図鑑の蝶」であることに気づく
3. **問い（Question）**: なぜこれを「蝶」と呼ぶのか？という根源的疑問が生じる
4. **変容（Transformation）**: 「見る」「名付ける」という行為自体への意識が変化する
